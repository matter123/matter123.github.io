Coding guidelines

symbol names

All symbols are to be in underscore_case. Systems Hungarian is forbidden, Apps Hungarian is acceptable. Symbol names in global, namespace, fille, class, struct, union, or function scope should be descriptive and only use full words and common acronyms and abbreviations.
