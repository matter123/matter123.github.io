copies upto num bytes from src to dest

copies upto num bytes from src to if a NULL byte is found when coping, then the rest of dest will be filled with NULL bytes. If the number of bytes remaining to copy is less then the length of the current char, then the current char is not copied and the rest of dest is filled with NULL bytes

Note
this function DOES NOT guarantee that dest will end with a NULL byte

srcpointer to a string of UTF-8 chars
