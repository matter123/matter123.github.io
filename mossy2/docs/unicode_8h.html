general purpose encoding conversion

converts a stream between two different encodings

Parameters
streaminput stream of bytes
in_leninput stream length
frombest guess of input stream encoding
todesired output encoding
out_streampointer to output stream of bytes
out_lenmax length of output stream
Note
if both out_stream and out_len are NULL and 0, respectively convert will allocate a stream of the required length, the caller is responsible to free the stream when done
Returns
number of bytes copied to out_stream if the values is larger than out_len, the output stream was not big enough to contain the entire input stream, and the value returned is how big the output stream needs to be
