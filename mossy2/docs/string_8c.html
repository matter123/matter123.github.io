copies upto num bytes from src to dest

copies upto num bytes from src to if a NULL byte is found when coping, then the rest of dest will be filled with NULL bytes. If the number of bytes remaining to copy is less then the length of the current char plus 1, then the current char is not copied and the rest of dest is filled with NULL bytes

Note
this function DOES guarantee that dest will end with a NULL byte
Parameters
destpointer to a block of memory at least num bytes big
srcpointer to a string of UTF-8 chars
numnumber of bytes to fill dest with
Returns
pointer equal to dest
